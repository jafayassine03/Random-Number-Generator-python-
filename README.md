Random Number Generator (python)

A simple python program that generates a random number between two numbers chosen by the user.

Description :

This program asks the user to enter a minimum number and a maximum number, then the program randomly select a number within that  range.

Features :

User input the min and max numbers

Generate a random numbers between two values chosen by the user
